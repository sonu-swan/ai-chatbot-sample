# 🤖 Artificial Intelligence Chatbot

A modern Java-based Artificial Intelligence Chatbot built using Java Swing. The application provides an interactive chat interface with rule-based responses and basic Natural Language Processing (NLP) techniques.

## ✨ Features

- 💬 Modern ChatGPT-inspired UI
- 🖥️ Java Swing GUI
- 🧠 Basic NLP (input preprocessing)
- 🤖 Rule-based AI responses
- 📅 Current Date & Time support
- ❓ Frequently Asked Questions (FAQ)
- ⚡ Fast and lightweight
- 🎨 Dark theme interface

## 🛠️ Technologies Used

- Java 17+
- Java Swing
- OOP Concepts
- Collections Framework
- Rule-Based NLP

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/ai-chatbot-sample.git
```

2. Open the project in VS Code / IntelliJ IDEA / Eclipse

3. Compile and run

```bash
javac src/*.java
java -cp src Main
```
